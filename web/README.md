# Static Frontend

This folder contains a minimal static frontend that can be deployed on Netlify or GitHub Pages.

## Local Use

Open `index.html` in a browser and point the API URL field to your FastAPI backend.

## Deployment

- Netlify:
  - Publish directory: `web`
- GitHub Pages:
  - Serve the `web/` directory through a Pages workflow or copy `index.html` into the publishing branch.
