# Paper Outline

## Working Title
Explainable Real-Time Field Plant Disease Detection and Severity Estimation Using MobileNetV3 and YOLO

## Core Story

1. Train on clean public data.
2. Test on real field data and show generalization gap.
3. Fine-tune on a small real-world dataset.
4. Demonstrate improved field performance.
5. Add lesion localization, severity estimation, and explainability.

## Main Experiments

- PlantVillage training and in-domain test
- Zero-shot field evaluation
- Field fine-tuning
- Field test after fine-tuning
- Optional YOLO lesion segmentation comparison
